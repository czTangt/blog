为了更好的使用电脑，这里记录一下关于 Windows 的相关配置。

电脑相关

屏幕鼠标切换

目前使用的是双屏，虽然想的是手不离开键盘，但是屏幕切换的时候总要调整当前鼠标的位置，不然焦点就在另一个屏幕上了。所以发现了当前的项目 MouseSwitcher，它可以通过快捷键 Alt + 向左/右键 进行鼠标的切换。虽然实验下来发现不能解决聚焦的问题，但是能切换鼠标就是进步，聚焦可以通过别的程序快捷键来实现。这里贴一下 介绍，可以看这个来进行掌握。

后期发现可以在 控制面板 -> 轻松使用设置中心 -> 使鼠标更易于使用 中进行配置，选择这个 通过将鼠标悬停在窗口上来激活窗口 就可以了。这样当我们在不同屏幕切换鼠标的时候，我们就可以直接激活聚焦于当前窗口了，快捷方便。

Edge

标签页切换

面对众多标签页，可以使用快捷键进行切换。其中从左到右是 Ctrl + Tab，从右到左是 Ctrl + Shift + Tab。而对于我的垂直页面，从上到下就是 Ctrl + Tab，从下到上就是Ctrl + Shift + Tab。同时可以使用 Ctrl + number 的方式来切换标签页，Ctrl + 9 就是最后一个，而其它的则是按顺序特定切换。

主页展示

使用 Ctrl + T 可以直接打开新的主页，同时这里配置了 青柠起始页 插件，使得主页直接聚焦于上方搜索栏中，这样就可以直接键盘输入内容进行搜索了。并且 Ctrl + N 为打开新的 edge 浏览器，可以实现新开。

其余操作

  • 可以使用 PhUp 和 PgDn(空格) 来替代鼠标的滚轮，实现上下页滑动的效果
  • Ctrl + L 和 Alt + D 都是选择地址栏中的 URL 以进行编辑
  • Ctrl + R 和 F5 作用一样，重新加载当前页，不过有缓存内容，加上 Shift 可以忽略缓存内容
  • Ctrl + W 关闭当前标签页，加上 Shift 就是关闭当前窗口了
  • Alt + 向左/右键 是返回/前进，但是我们在上面屏幕的切换中占用了这个快捷键，所以不能使用了

Vscode

终端与编辑框

这里 Terminal 的快捷键就是 Ctrl + '，因此可以通过这个和上面的鼠标设置达成在编辑框和终端切换聚焦的功能。只要打开终端，那么当前焦点就会在终端上，所以可以直接使用快捷键关闭终端显示，这样聚焦就在编辑框了，完成了焦点的切换。
